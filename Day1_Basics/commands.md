# Linux Practical - Day 1 Basics

## File & Directory Commands
- pwd → show current directory
- ls → list files
- mkdir test → create folder
- touch file.txt → create file

## Permissions
- chmod 755 file.sh
- chown user:user file.txt

## Process Management
- ps -ef
- top
- kill -9 PID

## Networking
- ping google.com
- netstat -tulnp
- ssh user@server-ip
